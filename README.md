# Bookmark Manager


## User Stories

```
As a web surfer
So that I can access a website I bookmarked
I want to be able to display a list of bookmarks
```

```
As a web surfer
So that I can revisit a webpage 
I want to be able to add a url to my list of bookmarks
```
